Kalahelm
========

Implementation of the game Kalaha (also known as Mancala or Kalah) in Elm.

Try it on http://janne.github.io/kalahelm/

Find [instructions here](https://www.fgbradleys.com/rules/Mancala.pdf).

Setup
-----

* `npm install`
* `elm package install`
* `npm start`
* `open http://localhost:3000`
